# Waste Management & Recycling Dashboard (Power BI)

## Project Overview
This project analyses waste generation and recycling trends in Singapore from 2003 to 2017 using Power BI. The goal is to explore sustainability performance and identify opportunities to improve recycling efficiency.

---

## Objectives
- Analyse trends in waste generation over time
- Evaluate recycling performance across years and waste categories
- Identify high-impact areas for improving sustainability outcomes

---

## Tools Used
- Power BI (dashboard design & visualisation)
- Excel / CSV (data source)

---

## Dashboard Features
- KPI cards: total waste generated, total recycled waste, average recycling rate
- Trend analysis: waste generation and recycling rate over time
- Category analysis: waste generation and recycling performance by waste type
- Interactive filtering by waste category

---

## Key Insights
- Waste generation has increased over time, indicating growing demand for waste management
- Recycling rates have improved significantly, showing progress in sustainability efforts
- Food waste contributes the largest share of total waste but has relatively low recycling rates
- Some materials (e.g. wood) achieve high recycling efficiency, highlighting potential best practices

---

## Business Implications
The analysis suggests that targeted improvements in high-volume, low-recycling categories (such as food waste) could significantly improve overall sustainability performance.
